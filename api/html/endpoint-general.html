Загальні правила¶

Для роботи з точками доступу проекту «Суд на долоні» необхідно вказувати такі заголовки запиту:

  • content-type - має бути встановлений як "application/json;charset=UTF-8"
  • Authorization - має містити JWT токен, що ідентифікує користувача та наданий після входу в обліковий запис.

Перший заголовок є обов’язковим. Заголовок для авторизації не є обов’язковим, але без нього система буде надавати лише базові дані.

Більше щодо отримання та оновлення JWT токену, дивіться розділ «авторизація користувача»
