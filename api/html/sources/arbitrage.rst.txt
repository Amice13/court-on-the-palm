Арбітражні керуючі
==================

Схема даних
~~~~~~~~~~~

:person:
    об'єкт, `фізична особа <person.html>`_
    
    Опис фізичної особи, яка виконує функції арбітражного керуючого.
    
:email:
   рядок
   
   Електрона пошта арбітражного керуючого (за наявності).

:phone:
   рядок
   
   Телефон арбітражного керуючого (за наявності).

:certificateNumber:
   рядок
   
   Номер свідоцтва арбітражного керуючого.
    
:certificateStatus:
   рядок
   
   Статус свідоцтва арбітражного керуючого (``діє`` або ``не діє``).
    
:arbitrageNumber:
   рядок
   
   Ідентифікатор арбітражного керуючого з Єдиного реєстру нотаріусів.

:updatedAt:
   рядок, `дата <date-format.html>`_

   Дата останнього оновлення.
