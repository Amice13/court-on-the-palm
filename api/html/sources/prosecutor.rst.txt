Прокурори
=========

Схема даних
~~~~~~~~~~~

:person:
    об'єкт, `фізична особа <person.html>`_
    
    Опис фізичної особи, яка виконує функції працівника прокуратури.
    
:address:
   об'єкт,`адреса <address.html>`_
   
   Поштова адреса за наявності.
    
:procuracyRegion:
    рядок
    
    Назва регіону відповідної прокуратури (назва області або загальноукраїнський регіон).
    
:declarations:
    масив об'єктів, декларація
    
    Клас валюти за українською національною статистичною класифікацією на основі `ISO 4217 <https://en.wikipedia.org/wiki/ISO_4217>`_ - 3 знаки.
    
:exchangedate:
    рядок, `дата <date-format.html>`_
    
    Дата вивантаження валютного курсу.

:updatedAt:
   рядок, `дата <date-format.html>`_

   Дата останнього оновлення.

Декларація
~~~~~~~~~~

Декларація про майновий стан та доходи громадян

Схема даних
-----------

:id:
    рядок
    
    Ідентифікатор декларації з НАЗК.

:year:
    рядок
    
    Рік подання декларації.
    
:kind:
    рядок
    
    Тип декларації.
