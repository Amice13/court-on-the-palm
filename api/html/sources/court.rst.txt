Суди
====

Схема даних
-----------

:courtCode:
    рядок
    
    Ідентифікатор судової установи у базах данних Державної судової адміністрації.

:courtName:
    рядок
    
    Повна назва судової установи у базах данних Державної судової адміністрації.

:courtRegion:
    рядок
    
    Область, у якій діє судова установа.

:instanceType:
    рядок
    
    Інстанція судової установи (``Перша``, `Апеляційна`, ``Касаційна``).

:openingHours:
    рядок
    
    Години работи судової установи (за наявності відповідної інформації).

:email:
   рядок
   
   Електрона пошта суду (за наявності).

:phone:
   рядок
   
   Телефон суду (за наявності).

:url:
   рядок
   
   Посилання на офіційний вебсайт судової установи.    

:organization:
    об'єкт, `юридична особа <organization.html>`_
    
    Опис юридичної особи судової установи.

:address:
   об'єкт,`адреса <address.html>`_
   
   Поштова адреса суду.    

:updatedAt:
   рядок, `дата <date-format.html>`_

   Дата останнього оновлення.
