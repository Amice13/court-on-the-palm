Державний реєстр виборців
=========================

База даних адрес виборців з відкритих данних `Державного реєстру виборців <https://www.drv.gov.ua/portal/!cm_core.cm_index>`_.

Одиниця даних - певний геонім (вулиця, площа, проспект тощо)

Інформація оновлюється один раз на півріччя.

Схема даних
-----------

:parentCode:
    рядок
    
    Код геоніму вищого рівня.
    
:regionCode:
    рядок

    Код регіону.
    
:ATOCode:
    рядок
    
    Код територіально-адміністративної одиниці.
    
:level:
    рядок
    
    Рівень територіально-адміністративної одиниці.
    
:code:
    рядок, `дата <date-format.html>`_
    
    Код геоніму.

:streetCode:
    рядок
    
    Код вулиці.

:postalCode:
    рядок
    
    Поштовий код адреси виборця.

:precinctNum:
    рядок
    
    Номер виборчої дільниці, до якої належить виборча адреса.

:districtNum:
    рядок
    
    Номер виборчого округу, до якого належить виборча адреса.

:region:
    рядок
    
    Область, в якій розташована виборча адреса.

:district:
    рядок
    
    Район/район у місті, в якому розташована виборча адреса.

:locality:
    рядок
    
    Населений пункт, в якому розташована виборча адреса.

:street:
    рядок
    
    Назва вулиці.

:oldDistrict:
    рядок
    
    Попередня назва району/району в місті, якщо відбувалася зміна назви.

:oldLocality:
    рядок
    
    Попередня назва населеного пункту, якщо відбувалася зміна назви.

:oldStreet:
    рядок
    
    Попередня назва вулиці, якщо відбувалася зміна назви.

:streetNumber:
    масив рядків
    
    Перелік номерів будинків, які є на відповідному геонімі.
