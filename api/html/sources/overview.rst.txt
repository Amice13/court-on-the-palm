Огляд
=====

API проекту "Суд на долоні" - це єдиний інтерфейс для баз даних проекту.

API в даному випадку розуміється як REST-ful інтерфейс, що надає програмний доступ до баз даних, які наявні в проекті "Суд на долоні". 
Він надає URL-адреси для доступу до ресурсів і використовує вбудовані HTTP особливості, щоб повертати відповіді на запит.

API приймає JSON вміст у запитах та повертає JSON вміст у всіх свої відповідях. Підтримується лише UTF-8 кодування і для запитів, і для відповідей.

Стабільність API
~~~~~~~~~~~~~~~~

API може змінюватися в певних деталях. Точки входу API будуть стабільними, проте формати запитів та відповідей можуть змінюватись. Окрім цього будуть з'являтися нові сутності в базах даних проекту.
