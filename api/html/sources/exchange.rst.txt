Курс валют
==========

Курс валют Національного банку України на певну дату.

Інформація оновлюється кожні 1-2 дні з `API Національного банку України <https://bank.gov.ua/NBUStatService/v1/statdirectory/exchange>`_.

Схема даних
-----------

:r030:
    рядок
    
    Код валюти або банківського металу, цифровий код валюти згідно з Класифікатором іноземних валют - 3 знаки.
    
:txt:
    рядок

    Повна назва валюти.
    
:rate:
    рядок
    
    Гривневий еквівалент за одну одиницю валюти.
    
:cc:
    рядок
    
    Клас валюти за українською національною статистичною класифікацією на основі `ISO 4217 <https://en.wikipedia.org/wiki/ISO_4217>`_ - 3 знаки.
    
:exchangedate:
    рядок, `дата <date-format.html>`_
    
    Дата вивантаження валютного курсу.
