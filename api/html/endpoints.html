Точки входу¶

Точки входу API проекту «Суд на долоні» надають окремий доступ до кожного з реєстрів, доступ до яких надається в проекті. А саме:

  • Єдиний державний реєстр судових рішень
  • Реєстр судів
  • Єдиний державний реєстр юридичних осіб
  • Єдиний Реєстр Адвокатів України
  • Реєстр атестованих судових експертів
  • Єдиний реєстр нотаріусів
  • Єдиний реєстр арбітражних керуючих

Окрім цього надається інформація щодо:

  • суддів
  • працівників прокуратури
