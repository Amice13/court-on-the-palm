Поштова адреса¶

Схема даних¶

source:

рядок

Неочищений рядок з повною адресою.
fullAddress:

рядок

Повна очищена поштова адреса, не включає номер приміщення.
apartment:

рядок

Номер приміщення (квартири, офісу тощо).
district:

рядок

Назва району або району в місті.
oldDistrict:

рядок

Попередня назва району або району в місті, якщо було перейменувано.
streetAddress:

рядок

Назва геоніму (включає також тип геоніму - вулиця, проспект, бульвар тощо).
oldStreetAddress:
 

рядок

Попередня назва геоніму, якщо було перейменувано.
locality:

рядок

Назва населеного пункту.
oldLocality:

рядок

Попередня назва населеного пункту, якщо було перейменувано.
region:

рядок

Назва області.
oldRegion:

рядок

Попередня назва області, якщо було перейменувано.
postalCode:

рядок

Поштовий індекс.
countryName:

рядок

Назва країни.
geo:

геоточка (довгота, широта)

Координата адреси (за наявності).
